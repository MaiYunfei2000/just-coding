## !注意事项!

程序编写环境为 MacOS 10.15.5，Matlab R2019a。

数据输出函数使用了官方推荐的内置函数`writecell`。由于兼容性问题，官方对于 2019 版本开始的用户不建议使用旧的函数如`xlswrite`等，详情参见官方文档相应页面。旧的函数无论在 Windows 上的 2019 版本是否会报错，在 MacOS 上是会报错的。

对于 2019 以下的版本，需要注释掉含有`writecell`的语句在运行（代码位于{run.m}很靠后的位置），否则跑完实验程序后在结束指导语或即将退出 PTB 前会报错而卡死。然后对于已经加载在命令行的变量数据，使用任意手段导出。
